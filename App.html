FIZZBUZZ: A coding Challenges

Let play a little guessing game! FIZZBUZZ is a JavaScript coding excerise using loops. Write a program which displays the numbers from 1 to a number of choice. Then display the Number that Divisible by FizzValue in Fizz and Divisible by BuzzValue in Buzz. When the number is Divisible by both it will display FizzBuzz.

  • displays the numbers from 1 to a number of choice
  • Number that Divisible by FizzValue are display as Fizz
  • Number that Divisible by BuzzValue are display as Buzz
  • Number that are Divisible by Both Value will display as FizzBuzz

Instructions

Enter a number Fizz Value and Buzz value.
Press the button, and see where the FizzBuzz is.
*Note*

  • Make sure the numbers are Positive
  • The Stop Value should always be greater then the Multiplication of FizzValue and BuzzValue
  • The Maximum number for stop is 9000
